Типы документов
===============

.. important:: Документы, помеченные "*" формирует оператор ЭДО при отправке документооборота.  

.. warning:: Остальне типы документов с ФНС и другими контролирующими органами, с которыми возможно работать с помощью API Контур.Экстерна, описаны в полной версии документации.
